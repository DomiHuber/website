# Genesis Block

The genesis block is the first block in the Bitcoin blockchain, mined by Satoshi Nakamoto on January 3, 2009. It has a block height of 0 and contains a message referencing a newspaper headline about bank bailouts. Its structure is hardcoded in every Bitcoin client.
