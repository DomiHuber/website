# Nonce

A nonce is a 32-bit field in the block header that miners vary when performing Proof-of-Work. By changing the nonce, miners attempt to find a valid hash below the difficulty target. When nonce space is exhausted, other fields such as the coinbase extra nonce are adjusted.
